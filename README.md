# 🚀 IntelliBuilder

AI-Powered Application Generator using a Compiler-Inspired Architecture

## Overview

IntelliBuilder converts natural language prompts into structured application configurations and interactive web applications.

Instead of manually designing pages, APIs, database schemas, roles, and business rules, users can simply describe the system they want, and IntelliBuilder automatically generates the application structure.

Example Prompt:

Build a Hospital Management System with patient registration, doctor management, appointments, billing, pharmacy and reports.

Generated Output:

* Dashboard
* Pages
* Navigation
* Roles
* Permissions
* Business Rules
* Database Tables
* API Endpoints
* Interactive Application UI

# Features

## AI Compiler Pipeline

The application follows a compiler-inspired architecture:

1. Intent Extraction
2. Module Discovery
3. Schema Generation
4. Validation
5. Repair
6. Runtime Generation

## Validation Engine

The Validation Engine checks generated configurations for:

* Missing Pages
* Missing Roles
* Missing APIs
* Missing Database Tables
* Invalid Structures

---

## Repair Engine

The Repair Engine automatically fixes incomplete configurations.

Example:

Missing Permissions

↓

Validation Detects Issue

↓

Repair Engine Generates Default Permissions

↓

Application Continues Successfully

# Installation

Clone Repository

```bash
git clone <repository-url>
cd IntelliBuilder
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Run Backend

```bash
python app.py
```

Run Frontend

Open:

```text
landing.html
```

using Live Server.
